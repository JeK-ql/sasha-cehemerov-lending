Виїзд знизу

Шторка виїжджає вгору як картка Apple Wallet, закриває ~95% екрана. Зверху видно затемнений сайт.

Поля під палець

Великі, без рамок — тонка лінія. Активна лінія підсвічується червоним. Тап по «Телефон» → цифрова клавіатура.

Нова Пошта

Місто — автопідказка при введенні. Відділення — список. Жодних індексів і домашніх адрес.

Оплата в шторці
